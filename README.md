Kaomoji Lab is a web application dedicated to the world of kaomoji. It provides a platform for users to discover, copy, and share a vast collection of over 6,000 kaomojis. The application is designed with a clean and intuitive user interface, making it easy for users to find the perfect kaomoji for any occasion.

## Features

- **Extensive Kaomoji Collection:** Access a library of over 6,000 kaomojis, organized into various categories and tags.
- **One-Click Copy:** Easily copy your favorite kaomojis to the clipboard with a single click.
- **Search and Filter:** Quickly find specific kaomojis using the search bar or by browsing through categories and tags.
- **Responsive Design:** Enjoy a seamless experience across all devices, from desktops to mobile phones.
- **Admin Panel:** A dedicated admin panel for managing kaomojis, categories, and tags. (Available for local use only)
